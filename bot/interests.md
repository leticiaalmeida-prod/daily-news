# Priority tiers

These map to the relevance categories the comprehension stage assigns
(must-read / relevant / tangential) — a Tier 1 match is usually must-read, a
Tier 3 match is usually tangential, but judge each article on its own merits
rather than mechanically by tier.

## Tier 1 — Gecko's direct territory (always surface)

- AI agents and infrastructure: MCP protocol developments, agent
  frameworks, API orchestration, agentic AI, tool use, function calling,
  LLM infrastructure, agent-to-API patterns. Anything where agents interact
  with external services — this is Gecko's core.
- API ecosystem: API management, documentation tools (Mintlify, Stainless,
  Postman), SDK generation, API security, developer experience platforms —
  Gecko's neighbors, competitors, or acquisition signals.
- Solana ecosystem: protocol upgrades, developer tooling, DeFi
  infrastructure, Superteam activity, Solana Foundation moves, on-chain
  payment protocols, x402 — anything touching the stack Gecko lives on.
- Credential security and agent safety: API key management, agent
  authentication, prompt injection, tool poisoning — anything about trust
  and safety in agentic systems. Gecko's anti-poisoning and keyless auth
  live here.

## Tier 2 — The strategic lens (surface when substantive)

- Stablecoins and payment rails: regulation, adoption, USDC, Circle,
  cross-border payments, pay-per-call models — relevant to x402 and how
  agents will pay for API access.
- Crypto regulation worldwide: SEC, CFTC, MiCA, Brazilian CVM, any country
  legislating crypto or AI agents — policy shifts that affect builders.
- VC and fundraising: notable raises in crypto infrastructure, AI
  infrastructure, developer tools, API platforms — especially seed and
  Series A (Gecko's stage). Acquisitions in the space (Anthropic/Stainless,
  Workday/Pipedream) are high signal.
- Developer tools and distribution: open source adoption strategies,
  developer-first go-to-market, bottom-up SaaS, PLG motions — how companies
  win developers for free and monetize the provider. Gecko's business model
  in other clothing.

## Tier 3 — The bigger picture (surface selectively)

- Global politics and finance: geopolitical shifts, trade policy, central
  bank decisions, inflation — anything that would affect tech funding,
  crypto markets, or emerging markets (especially Brazil and Latin
  America).
- Technology broadly: major AI model releases, platform shifts, big tech
  infrastructure moves — the landscape Gecko operates in.
- Crypto market movements: only notable structural shifts — a major token
  moving 10%+, a liquidity event, a market-wide turning point. Not daily
  price noise.
